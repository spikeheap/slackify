### Installation

Add a `.env` with your credentials:

```
# Spotify credentials
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=

# Docker DB credentials
POSTGRES_HOST=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=

# For secure Rack cookies
SESSION_SECRET=
```
